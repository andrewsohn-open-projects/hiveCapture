HiveCapture Screen Capture Batch Tool
=========
Windows Application Screen Capture CSV Batch Tool presented by Hivelab Corp. Samsung Opt Team.


Requires
------------------
 * `electron 1.4.1`
 * `jquery 3.1.1`
 * `underscore 1.8.3`
 * `electron-json-storage 2.0.1`
 * `async 2.1.2`
